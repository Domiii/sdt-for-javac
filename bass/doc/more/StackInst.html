Stack Manipulation Instructions

mnemonicargumentspoppushdescription
pop vmspec 1-pop top element
pop2 vmspec 2 1-pop top 2 elements
dup vmspec 11 1duplicate top element
dup_x1 vmspec 2 11 2 1duplicate top element; put 2 down
dup_x2 vmspec 3 2 11 3 2 1duplicate top element; put 3 down
dup2 vmspec 2 12 1 2 1duplicate top 2 elements
dup2_x1 vmspec 3 2 12 1 3 2 1duplicate top 2 elements; put 3 down
dup2_x2 vmspec 4 3 2 12 1 4 3 2 1duplicate top 2 elements; put 4 down
swap vmspec 2 11 2swap top 2 elements

The numbers 1, 2, 3 or 4 on pop/push columns represents one "unit" of operans stack. JVM opetand stack has an associated depth, where a value of type long or double contributes two units to the depth and a value of any other type contributes one unit. Stack manipulation instructions cannot be used to modify or break up individual values. Please see vmspec Sec. 3.6.2.

Examples

ldc 1    ; stack -- 1
ldc 2L   ; stack -- 1, 2L
dup2_x1  ; stack -- 2L, 1, 2L
pop2     ; stack -- 2L, 1
ldc 3    ; stack -- 2L, 1, 3
swap     ; stack -- 2L, 3, 1
pop      ; stack -- 2L, 3
dup_x2   ; stack -- 3, 2L, 3
