Bass language

Bass programs consist of a sequence of line-separated statements. Line separator is either a carriage-return character ('\r', 0x0D) or a newline character ('\n', 0x0A). In addition, a carriage-return character followed immediately by a newline character is treated as a single line separator.

There are three types of statement:

  • directives
  • instructions
  • labels

Directives and instructions can take parameters. These parameters are placed on the same line as the directives or instructions, separated by white spaces (0x00 through 0x20, excluding 0x0D and 0x0A).

Comment

Format

; any text

Description

A Bass comment is written with a semicolon (;), and it lasts until the end of the line.

A semicolon appeared within a quoted string is treated as an ordinary character.

Examples

; This is a comment
.package java.lang    ; This is a package declaration
.import java.io.*;    ; The last meaningful character of this line is "*"
ldc "123;456"         ; load a String with 7 characters

Label

Format

label :

Description

Bass labels are used to indicate Bass instructions. They are placed within methods.

label should be a simple names. Labels in the same method should be different.

Examples

    iload #1
    ifle L1
    ...
    goto L2
L1:
    ...
L2:
    ...
