Local int Variable Access Instructions

mnemonicargumentspoppushdescription
iload_0 vmspec -Iload local int variable #0
iload_1 vmspec -Iload local int variable #1
iload_2 vmspec -Iload local int variable #2
iload_3 vmspec -Iload local int variable #3
iload vmspec#var1-Iload local int variable #var1
wide-iload vmspec#var2-Iload local int variable #var2
istore_0 vmspec I-store local int variable #0
istore_1 vmspec I-store local int variable #1
istore_2 vmspec I-store local int variable #2
istore_3 vmspec I-store local int variable #3
istore vmspec#var1I-store local int variable #var1
wide-istore vmspec#var2I-store local int variable #var2

var1 is an un-signed integer (range: 0~28-1), var2 is an un-signed integer (range: 0~216-1). "#" is optional. White spaces between "#" and var1 or var2 are not permitted.

    Generic version

For programmer's convenience, Bass supports the following generic instructions. Bass will automatically determine which instruction is the most suitable.

mnemonicargumentspoppushdescription
iload#var2-Iload local int variable #var2
istore#var2I-store local int variable #var2

var2 is an un-signed integer (range: 0~216-1). "#" is optional. White spaces between "#" and var2 are not permitted.

Examples

iload_0         ; load local variable #0, treat it as an int
iload #4        ; load local variable #4, treat it as an int
wide-iload 256  ; load local variable #256, treat it as an int

var2 is an un-signed integer (range: 0~216-1). "#" is optional. White spaces between "#" and var2 are not permitted.

Examples

                ; treat them together as a long
lload #4        ; load local variable #4 & #5,
                ; treat them together as a long
wide-lload 256  ; load local variable #256 & #257,
                ; treat them together as a long
