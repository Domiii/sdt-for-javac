SOPRANO -- bytecode disassembler

Soprano is a disassembler to translate Java bytecode (.class file) to the corresponding bass program (.bass file). The name of the assembler is bass, so the name of the disassembler, of course, is soprano.

Synopsis

soprano [ options ] classFiles

Arguments may be in any order.

options
Command-line options.

classFiles
One or more class files to be disassembled.

Options

-d directory
Set the destination directory for bass files. The value directory can be absolute or relative to the current working directory. If -d is not specified, the destination directory is the current working directory.

-b
Produce output beiefly. For example, the .extends directive will be omitted for a class whose super class is java.lang.Object. By default, the output is produced in brief.

-b:none
Produce output detailedly.

-g
Translate instructions into generic forms. For example, instruction "03" will be translated into ldc 0. By default, the generated instructions are generic.

-g:none
Don't translate instructions into generic forms. For example, instruction "03" will be translated into iconst_0.

-i:package
Generate .import directives. The imported entities are packages. In this case, simple names are sufficient to repesent classes. This is the default setting for -i.

-i:class
Generate .import directives. The imported entities are classes. In this case, simple names are sufficient to repesent classes.

-i:none
Don't generate .import directives. In this case, classes will be represented by fully qualified names.

Examples

soprano String.class reflect\Field.class

soprano -d myjavac\classes -b:none -g:none -i:none *.class
